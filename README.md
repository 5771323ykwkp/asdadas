# 项目简介

基于SpringBoot3+Vue3的医疗服务系统，是一款集成了用户、医生、患者、科室、预约、就诊记录、处方、药品及排班管理等模块的医疗信息管理系统。

## 核心功能

- **用户模块**
  - 用户登录、注册
  - 密码修改、找回
  - 用户增删改查
  - 根据角色、用户名查询用户
  - 用户状态修改
  - 分页查询用户

- **医生模块**
  - 医生增删改查
  - 医生与用户绑定、解绑
  - 医生状态修改
  - 根据科室、用户ID查询医生
  - 分页查询医生

- **患者模块**
  - 患者增删改查
  - 患者注册
  - 患者与用户绑定、解绑
  - 批量删除患者
  - 分页查询患者

- **科室模块**
  - 科室增删改查
  - 分页查询科室

- **预约模块**
  - 预约新增、取消、完成、修改
  - 根据患者、医生查询预约
  - 分页查询预约

- **就诊记录模块**
  - 就诊记录增删改查
  - 根据患者、医生查询就诊记录
  - 分页查询就诊记录

- **处方模块**
  - 处方新增、状态修改
  - 处方明细增删改查
  - 根据患者、医生、就诊记录查询处方
  - 分页查询处方

- **药品模块**
  - 药品增删改查
  - 药品库存、状态修改
  - 根据分类查询药品
  - 分页查询药品

- **药品分类模块**
  - 药品分类增删改查
  - 分页查询药品分类

- **排班模块**
  - 排班增删改查
  - 批量新增排班
  - 排班状态修改
  - 根据医生、日期查询排班
  - 分页查询排班

- **文件模块**
  - 文件上传
  - 多文件上传

## 适用场景

适用于医疗机构内部管理，包括用户权限管理、医生患者信息管理、预约就诊流程管理、处方及药品管理、科室及排班管理等。

## 技术特点

- 后端：基于SpringBoot3、MyBatisPlus、JWT、SpringSecurity等技术实现。
- 前端：采用Vue3、Element Plus、Node.js、Axios等。
- 数据库：使用MySQL8。
- 系统架构：前后端分离，易于维护与扩展。
